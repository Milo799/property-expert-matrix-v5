## ⑦ 环境投诉闭环
```
投诉受理 → 分类定级(一般/紧急/重大) → 责任区域定位(GIS网格)
  ↓
派单整改(限时: 一般4h/紧急2h/重大30min) → 现场拍照上传
  ↓
质检复核 → 满意度回访(≥4星闭环) → 月度归因分析
```
> 时效指标: `闭环率 = 按时关闭数/总投诉数 ≥ 95%` | `复发率 = 7d内同类投诉占比 ≤ 5%`
