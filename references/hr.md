# 人力资源专家知识卡片 v5
## 1. 潮汐排班模型
| 变量 | 公式/规则 |
|---|---|
| 需求预测 | `D(t)=α·H(t)+β·S(t)+γ·W(t)` (H=历史客流 S=季节 W=天气) |
| 弹性人力池 | `P=P₀+ΔP·σ(t)` (σ潮汐系数0.6~1.8) |
| 排班优化 | `min Σ(加班费+闲置成本)` s.t. `Σshift≥D(t)`, `工时≤法定` |
| 峰谷比 | `R=Peak/Valley`, R>2.5→小时工; R<1.5→标准班 |
| 四班三运转 | 早08-16/中16-24/夜00-08, 7天轮换, 间隔≥11h |
## 2. 绩效考核体系
| 维度 | 权重 | KPI指标 |
|---|---|---|
| 业绩结果 | 40% | 营收达成率/成本节约/工单完结率 |
| 过程质量 | 25% | SOP执行度/客户满意度/差错率 |
| 能力成长 | 15% | 技能认证数/培训出勤/跨岗能力 |
| 协作贡献 | 20% | 团队评分/创新提案/知识分享 |
**评分**: `S=Σ(Wi×Ki)` → ≥95:A / 80-94:B / 60-79:C / <60:D | **分布**: A:15% B:50% C:30% D:5%
## 3. 外包人员管理
| 环节 | 管控要点 |
|---|---|
| 准入审查 | 资质三证+信用≥B+物业案例≥3个 |
| SLA签约 | 响应时效/完工标准/违约扣款/退出机制 |
| 过程管控 | 日报+周评+月审; 不合格>5%→限期整改 |
| 考核结算 | `结算=基准价×(1±KPI浮动)`, 扣分项明确 |
| 风险防范 | 禁事实劳动关系; 安全责任书; 雇主责任险 |
## 4. 薪酬福利框架
| 层级 | 结构 | 占比/说明 |
|---|---|---|
| 基础薪酬 | 基本工资+岗位津贴 | 60-70%, 对标市场P50 |
| 绩效薪酬 | 月/季/年度奖金 | 15-25%, 与评级挂钩 |
| 长期激励 | 期权/利润分享/跟投 | 5-10%, 核心人才专属 |
| 法定福利 | 五险一金/年假/高温 | 强制合规, 按当地标准 |
| 补充福利 | 商保/体检/餐补/节日 | 弹性积分制, 预算管控 |
## 5. 培训与发展
| 类型 | 对象/周期 | 评估 |
|---|---|---|
| 入职培训 | 新员工/7-14天 | 通关考试≥80分 |
| 技能提升 | 在岗/季度 | 实操考核+认证 |
| 管理培训 | 储备干部/半年 | 360评估+沙盘 |
| 特种培训 | 安全设备岗/年度 | 取证复审 |
**ROI**: `收益=(Δ绩效×人数×薪酬)-(培训费+工时损失)` | **九宫格**: X绩效 × Y潜力 → 明星/骨干/观察/淘汰
## 6. 劳动风控
| 风险 | 预警指标 | 应对 |
|---|---|---|
| 违法解除 | 无证据/未走程序 | 证据链+工会程序+N+1预案 |
| 工时争议 | 月加班>36h/无考勤 | 电子考勤+审批流+超时预警 |
| 薪酬纠纷 | 低于最低/拖欠 | 月审+工资条签收+调解机制 |
| 工伤事故 | 未缴社保/未培训上岗 | 签字+24h申报+巡查台账 |
| 竞业保密 | 离职带走客户 | 竞业协议(补偿≥30%)+权限回收 |
## 7. 人效管控模型
| 指标 | 公式 | 物业基准 |
|---|---|---|
| 人均面积 | `总面积÷在岗人数` | 住宅8000-12000㎡/人 |
| 人均营收 | `总营收÷平均在岗` | 8-15万/人/年 |
| 人工成本率 | `人工总额÷总营收` | 40-55% |
| 工时利用率 | `有效工时÷排班工时` | ≥75% |
| 人效提升 | `Δ产出/Δ投入-1` | 年目标+5-8% |
**杠杆**: ①机械化 ②一专多能 ③排班优化 ④外包 ⑤数字化
## 8. 员工满意度与流失率
| 指标 | 公式 | 健康区间 |
|---|---|---|
| 满意度指数 | `(Σ得分÷满分)×100%` | ≥75% |
| 年度流失率 | `离职÷((期初+期末)/2)` | 基层<25% 管理<10% |
| 主动流失率 | `主动离职÷总离职` | <70% |
| 新员工流失 | `≤90天离职÷同期入职` | <15% |
**满意度权重**: 薪酬30%+发展25%+管理20%+环境15%+文化10%
**预警**: `P=σ(α·薪酬偏差+β·满意度降+γ·司龄<1+δ·无晋升)`
**留人**: 关键人才1v1 → 个性化IDP → 3/6/12月追踪
