# IoT 智慧物联 — 专家知识卡片 v5

## ① 智慧通行
| 场景 | 策略 |
|------|------|
| 断网应急 | 本地缓存白名单 → 离线抬杆 → 网络恢复自动补传记录 |
| 月卡/临停 | 月卡≈白名单直放；临停=入场计时→出场计费→欠费拦截 |
| 防跟车 | 地磁双线圈+视频AI识别：1车1杆，尾随抓拍+声光驱离 |

## ② 消防联动（标准公式）
`水压↓ / 烟感=ON → 声光报警 → 切非消防电 → 电梯迫降首层 → 开排烟风机 → 启消防广播`
- 联动触发≤3s，全链路≤30s；消防指令优先级=最高，覆盖一切自动策略。

## ③ 设备智控
| 设备 | 运行策略 | 预防性维保联动 |
|------|----------|----------------|
| 电梯 | 高峰群控调度 / 平峰单梯运行 | 运行时长≥阈值 → 自动生维保单 → 年检到期预警 |
| 水泵 | 变频恒压 / 一用一备自动切换 | 启停次数↑ / 振动异常 → 预判轴承磨损 |
| 照明 | 照度补偿 + 人体感应 | LED光衰检测 → 批量更换提醒 |

## ④ 能耗智控
| 模块 | 机制 |
|------|------|
| 远传表计 | 水/电/气定时采集 → 峰谷分析 → 异常用量告警（突增>30%触发） |
| 公区节能 | 声光控(车库) + 定时(楼道) + 照度联动(外立面) |
| 节能公式 | `节电率 = (基准能耗 − 优化能耗) / 基准能耗 × 100%` |

## ⑤ 边缘计算架构
```
云端(策略下发/数据分析)
  ↕ 心跳30s
边缘节点(规则引擎/本地自治) ← 断网→切换自治模式，本地策略持续运行
  ↕ 直连
设备层(传感器/执行器/控制器)
```
- 自治核心：规则引擎常驻本地，断网≥5min自动切换；网络恢复后差异数据同步。

## ⑥ 数据安全
| 层级 | 措施 |
|------|------|
| 设备认证 | 一机一密 / 证书双向认证 / 固件签名校验 |
| 传输加密 | MQTT over TLS / HTTPS / 国密SM2/SM4可选 |
| 隐私保护 | 人脸/车牌数据脱敏存储 → 到期自动清理 → 业主可查可删 |

## ⑦ 应急响应（降级矩阵）
| 故障 | 降级机制 | 恢复条件 |
|------|----------|----------|
| 断电 | UPS供电(核心设备≥2h) → 道闸手动摇柄 → 电梯就近停靠开门 | 市电恢复 |
| 断网 | 边缘自治继续 → 本地记录缓存 → 恢复后增量同步 | 网络连通 |
| 设备故障 | 自动切换备用设备 → 告警上报 → 生成维修工单 | 更换/修复 |

## ⑧ 业务集成闭环
`IoT告警 → 规则过滤(去重/分级) → 工单生成(P0/P1/P2) → 派单(就近/专业匹配) → 现场处置 → 拍照回填 → 审核关单 → 数据归档`
- 闭环SLA：P0≤15min / P1≤30min / P2≤2h；超时自动升级。
