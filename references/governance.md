# 社区治理专家知识矩阵 v5

## ① 业委会全生命周期
| 阶段 | 触发条件 | 核心动作 | 输出物 | 时限 |
|------|---------|---------|--------|------|
| 筹备 | ≥20%业主联名/街道指定 | 成立筹备组→公示成员 | 筹备组名单 | ≤90d |
| 选举 | 筹备组完成候选人提名 | 双过半投票(人数+面积) | 选举决议 | ≤30d |
| 备案 | 选举通过后 | 街道备案+刻章 | 备案证明 | ≤30d |
| 履职 | 备案完成 | 制定规约/监督物业/管理公共收益 | 年度工作报告 | 持续 |
| 换届 | 任期届满(3-5年) | 重新选举→工作交接 | 新委员会 | ≤60d |
| 解散 | 不足半数/重大违规/业主大会决议 | 清算→移交档案→注销 | 注销证明 | ≤90d |

## ② 业主大会投票机制
- **法定人数**: 专有部分占比≥50% 且 人数≥50% 方可召开
- **表决规则**: 
  - 一般事项: 双过半 ✓ (面积>50% ∧ 人数>50%)
  - 重大事项: 双2/3 ✓ (面积≥2/3 ∧ 人数≥2/3)
  - 特别事项: 双3/4 ✓ (面积≥3/4 ∧ 人数≥3/4)
- **线上投票**: 区块链存证(哈希上链) + 实名认证 + 时间戳 + 不可篡改
- **公式**: `Q = Σ(投票面积) / Σ(总面积) ∧ Qₚ = Σ(投票人数) / Σ(总人数)`

## ③ 红色物业框架
`党委(引领)◄►业委会(自治)◄►物业(服务) → 三方协同议事平台(收集→协商→决议→督办)`
- **议事规则**: 月度联席 | 一事一议 | 决议公示≥7d | 街道列席

## ④ 公共收益管理
| 收益来源 | 占比(约) | 分配公式 |
|---------|---------|---------|
| 广告(电梯/外墙/道闸) | 35% | 70%业主基金 / 20%物业成本 / 10%社区公益 |
| 停车费(公共区域) | 25% | 同上 |
| 场地租赁 | 15% | 60%业主基金 / 30%物业成本 / 10%社区公益 |
| 其他(快递柜/充电桩) | 25% | 同上 |
- **公示**: 每季度公示(30d) → 审计年报 → 业主大会审议
- **专户**: 独立账户 | 双签审批 | 专款专用

## ⑤ 维修资金管理
```
申请 → 街道初审 → 双2/3表决 → 审批 → 招标 → 施工 → 验收 → 审计 → 拨付
  ▲                                                                  │
  └───────────────────────── 质保金(5%) ←────────────────────────────┘
```
- **双2/3表决**: 专有部分占比≥2/3 且 人数≥2/3 同意
- **紧急通道**: 电梯故障/消防隐患/外墙脱落 → 绿色通道(48h响应)
- **拨付**: 30%预付 + 50%进度 + 15%验收 + 5%质保金(1年后)

## ⑥ 物业服务质量评估
| 维度 | 权重 | 评估方式 | 合格线 |
|------|------|---------|--------|
| 业主满意度 | 40% | 问卷≥50%回收率 | ≥80分 |
| 第三方评估 | 30% | 神秘顾客+现场核查 | ≥75分 |
| 街道考核 | 20% | 年度综合评定 | ≥70分 |
| 投诉处理率 | 10% | 24h响应/72h闭环 | ≥90% |
- **综合评分**: `S = 0.4×C + 0.3×T + 0.2×G + 0.1×R`
- **奖惩**: S≥90 续聘优先 | 70≤S<80 整改期 | S<70 启动解聘

## ⑦ 社区矛盾调解机制
```
预警(网格员巡查) → 微调解(楼栋长) → 正式调解(业委会+物业+街道)
       ↓ 7d未解              ↓ 15d未解              ↓ 30d未解
   升级:社区调解委员会 ──→ 街道仲裁 ──→ 人民调解委员会 ──→ 法院诉讼
```
- **升级路径**: 调解(免费) → 仲裁(低成本) → 诉讼(司法终局)
- **调解协议**: 具有民事合同效力 → 可司法确认 → 强制执行
- **时限**: 简易纠纷≤7d | 一般纠纷≤15d | 复杂纠纷≤30d
