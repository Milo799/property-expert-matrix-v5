## 💰 C2 租金计算引擎(6模式)
| 模式 | 公式 | 适用场景 |
|---|---|---|
| 固定租金 | R = A × r × 12 (A=㎡, r=元/㎡/月) | 标准商铺/写字楼 |
| 提成租金 | R = max(0, S×ρ − C₀) (S=营业额, ρ=扣率) | 餐饮主力/百货 |
| 保底取高 | R = max(R固, S×ρ) | 购物中心/品牌连锁 |
| 阶梯租金 | R = A×r₁(S≤T₁) ∥ A×r₂(T₁<S≤T₂) ∥ A×r₃(S>T₂) | 培育期商户 |
| 纯提成 | R = S×ρ (ρ=8~25%) | 快闪店/联营 |
| 推广费 | P = A×p + S×φ (p=固定费, φ=分摊率) | 大型MALL |
## ⚖️ 🆕 C3 租赁合同法务(民法典)
| 合规项 | 法规 | 处理流程/要点 |
|---|---|---|
| 买卖不破租赁 | §725 | 产权变更≠合同终止 → 新房东承继至期满 → 书面通知(签收留痕) |
| 优先购买权 | §726 | 出售前≥15日通知 → 租户15日内答复 → 逾期=放弃 → EMS回执/公证送达 |
| 装修残值归属 | §715 | 期满归出租方(不补偿); 提前解约:残值=原值×(剩余月/总月)×折损系数 |
| 违约金&押金 | §585 | 违约金≤损失30%; 抵扣顺序:欠租→违约金→损坏 → 超额另行主张 |

## 📈 C5 商业运营指标
| 指标 | 公式 | 基准 |
|---|---|---|
| 坪效 | Sales / 租赁面积(㎡) | 餐饮≥2000元/㎡/月; 零售≥1500 |
| 租售比 | 租金/营业额 × 100% | 餐饮10~15%; 零售15~20%; 娱乐≤10% |
| NOI | 总租金 − 运营成本 − 税费 | NOI率≥55%健康 |
| WALT | Σ(剩余租期ᵢ×租金ᵢ) / Σ租金ᵢ | 写字楼≥5年; 零售≥3年 |
| 空置率 | 空置面积 / 可租总面积 × 100% | <5%优秀; 5~10%正常; >10%🚩 |
