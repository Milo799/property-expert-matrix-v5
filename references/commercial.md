# 🏢 商业招商专家知识库 v5.0

## 📊 卡片1 | 招商线索漏斗模型
| 阶段 | 关键动作 | 转化率基准 | 预警阈值 |
|---|---|---|---|
| 线索→有效 | 资质初审/需求匹配 | ≥40% | <30% 🚩 |
| 有效→洽谈 | 方案报价/实地带看 | ≥60% | <50% 🚩 |
| 洽谈→意向 | 条款磋商/意向书 | ≥50% | <40% 🚩 |
| 意向→签约 | 合同签署/保证金 | ≥70% | <60% 🚩 |
> **全链路转化率** = 签约数/线索总数 × 100%；行业均值 6~12%

## 💰 卡片2 | 租金计算引擎（6模式）
| 模式 | 公式 | 适用场景 |
|---|---|---|
| 固定租金 | `R = A × r × 12` (A=面积, r=元/㎡/月) | 标准商铺/写字楼 |
| 提成租金 | `R = max(0, S × ρ - C)` (S=营业额, ρ=扣率, C=成本) | 百货/餐饮主力店 |
| 保底取高 | `R = max(R固定, S × ρ)` | 购物中心/品牌连锁 |
| 阶梯租金 | `R = A × r₁(S≤T₁) / A×r₂(T₁<S≤T₂) / A×r₃(S>T₂)` | 培育期商户 |
| 纯提成 | `R = S × ρ` (ρ通常8~25%) | 快闪店/联营模式 |
| 推广费 | `P = A × p + S × φ` (p=固定推广费, φ=营销分摊率) | 大型MALL统一营销 |

## 🔍 卡片3 | 营业额稽核模型
**POS对接** → 实时采集(Sales流水/退款/折扣) → 数据清洗(去重/异常值过滤)
```
异常检测算法:
① 同比偏差: |S_t - S_t-7| / S_t-7 > θ₁(默认30%) 🚩
② 环比断崖: S_t / S_t-1 < θ₂(默认50%) 🚩
③ 租售比异常: (R/S) < ρ_min 或 > ρ_max → 触发人工稽核
④ 零申报连续: S_t = 0, ∀t∈[t₀,t₀+n], n≥3 → 现场核查
⑤ 聚类离群: IsolationForest(score < -0.5) → 标记复核
```

## 📈 卡片4 | 商业运营指标体系
| 指标 | 公式 | 行业基准 |
|---|---|---|
| 坪效 | `Sales / 租赁面积(㎡)` | 餐饮≥2000元/㎡/月；零售≥1500 |
| 租售比 | `租金 / 营业额 × 100%` | 餐饮10~15%；零售15~20%；娱乐≤10% |
| NOI | `总租金收入 - 运营成本 - 税费` | NOI率≥55%健康 |
| WALT | `Σ(剩余租期×租金) / Σ租金` | 写字楼≥5年；零售≥3年 |
| 空置率 | `空置面积 / 可租总面积 × 100%` | <5%优秀；5~10%正常；>10%预警 |

## 📋 卡片6 | 免租期管理 + 租金递增
| 规则项 | 标准 |
|---|---|
| 免租期上限 | ≤合同总期10%（装修期通常1~3月） |
| 免租覆盖 | 仅免固定租金；物业费/推广费照常 |
| 提前解约 | 未满免租期退出→追缴全额免租金额 |
| 租金递增 | `R_n = R₀ × (1 + g)^(n-1)` (g=年递增率3~8%) |
| 递增触发 | 按周年日/自然年首月生效 |

## ⭐ 卡片7 | 商户信用评级模型（5维度）
| 维度 | 权重 | 评分标准(1-10分) |
|---|---|---|
| 经营能力 | 30% | 坪效达标率/营业额增长率/品牌等级 |
| 履约记录 | 25% | 租金准时率/投诉率/违规次数 |
| 财务健康 | 20% | 资产负债率/现金流/纳税评级 |
| 协同配合 | 15% | 营销活动参与度/数据报送及时率 |
| 品牌形象 | 10% | 消费者评分/媒体口碑/行业奖项 |
> **总分≥85**=A级(优先续租/优惠); **70~84**=B级(正常); **<70**=C级(观察/淘汰)
