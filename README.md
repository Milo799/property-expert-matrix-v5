## 🏢 项目简介

本项目是一个**企业级多智能体 (Multi-Agent) 业务专家技能库**，专为物业管理行业数字化转型设计。团队汇聚了 16 位拥有 10 年以上头部物业企业（如万物云、龙湖、碧桂园等）实战经验的专家角色。

**核心升级 (V5.2)**:
*   🛡️ **合规全景库**: 扩充至 **50+ 种舞弊手法**（吃空饷/阴阳合同/废料私卖等），覆盖 10 大业务线条。
*   💰 **业财一体**: 增强财务专家 (FIN)，支持 **SAP/ERP 凭证映射**、复杂商业计费 (保底取高/提成)。
*   📉 **数据治理**: 增强资产专家 (ASSET)，主导历史数据清洗、主数据标准与计费追溯。
*   🔋 **IoT 断网自治**: 增强 IoT 专家 (SMART)，制定断网/断电下的降级策略与本地缓存方案。
*   📝 **BRS 强制记录**: 业务需求讨论**必须**由 BRS 全程旁听，输出工业级标准需求文档。
*   📚 **DOC 100% 同步与冲突检测**: 任何技能变更自动触发全员冲突扫描、文档全量更新与 GitHub 推送。README 必须包含变更简述，杜绝遗漏。

## 🚀 核心工作机制

### 1. 🔥 强制圆桌会议 (Roundtable Discussion)
任何业务需求、方案评审、风险排查，**必须**通过多专家圆桌讨论。拒绝单点拍脑袋决策。

### 2. 📝 BRS 全程记录 (Documentation)
专属记录员全程旁听，将讨论转化为**可落地需求文档**。
*   **格式**: 包含背景目标、SOP 流程、字段定义 (输入/输出)、非功能需求、行业对标、**讨论注解**。
*   **兜底**: 若需求不明确，输出《圆桌会议纪要》，明确行动项 (Action Items)。

### 3. 🛡️ 风险雷达前置 (Risk Radar)
每位专家内置 Top 5 管理风险库 (如吃空饷、阴阳合同、私收费、虚假维保)。在方案设计阶段即进行防腐审查。
